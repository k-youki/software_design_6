ターン制MOBA(Multi Online Battle Arena)

#概要
・2on2で行う陣取りゲーム

・最終的に相手の本拠地を破壊することで勝利

・道中侵攻を阻む形で壁が設置されている

　・壊すと奥へ進めるようになる

##主な関数・変数となりえる物
・操作キャラ*4体(敵味方含む)

・壁*4枚(敵味方分)

・本拠地*2柱(敵味方分)

・フィールド

・サイコロ

####操作キャラ
・1ターンに一度サイコロを振り、移動・攻撃・その両方を行う

・移動か攻撃のみの場合はサイコロの出た目だけ移動かダメージを与える

・両方を行う場合は移動・攻撃した分をサイコロの出た目から引き、残りの分を他の行動に回す

####壁
・本拠地まで一定感覚でチーム毎に2枚ずつ設置される

・味方のみ擦り抜けが可能

・敵の壁は破壊しないと通過が不可

・一定の耐久値を持つ

####本拠地
・敵味方の陣地最奥に存在

・一定の耐久値を持つ

・破壊されると負け

####フィールド
・横長の長方形
